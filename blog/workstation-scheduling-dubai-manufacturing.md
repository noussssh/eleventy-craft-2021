## Scheduling Strategies

### Forward Scheduling

Start from today, schedule ahead:

- Release date is now
- Calculate when each operation completes
- Determine due date achievability
- Good for make-to-stock

### Backward Scheduling

Start from due date, work backward:

- Due date is fixed
- Calculate when each operation must start
- Determine if schedule is feasible
- Good for make-to-order

### Priority-Based

Sequence by importance:

- Customer priority
- Order value
- Due date urgency
- Strategic importance

### Shortest Processing Time

Maximize throughput:

- Quick jobs first
- Reduce average completion time
- Good for bottleneck stations

### FIFO

Fair and simple:

- First come, first served
- Predictable for customers
- Easy to manage
