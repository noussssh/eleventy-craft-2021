## Common Mistakes

### Scope Creep

Expanding scope:

*Problem*: Adding requirements mid-project
*Prevention*: Clear scope, change control

### Insufficient Training

Undertrained users:

*Problem*: Users don't know how to use system
*Prevention*: Adequate training investment

### Data Migration Issues

Bad data in new system:

*Problem*: Garbage in, garbage out
*Prevention*: Data cleansing, validation

### Customization Excess

Over-customizing:

*Problem*: Complex, hard to maintain
*Prevention*: Standard first, customize when necessary

### Timeline Compression

Rushed implementation:

*Problem*: Quality suffers
*Prevention*: Realistic planning
